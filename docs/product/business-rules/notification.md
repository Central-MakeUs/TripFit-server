# Notification (알림) 비즈니스 규칙

> 기획자 전달 알림 명세 표(2026-07-23)를 SSOT로 반영. MVP 푸시·카카오 공유는 **wave 3**. 스케줄러 정기 발송은 **wave 4**.

## 알림 이벤트 요약

| 규칙 ID | 이벤트 | 수신 대상 | 발송 조건 |
| :--- | :--- | :--- | :--- |
| **BR-NOTI-001** | 참여 완료 | 방장 | 참여자가 일정 제출을 완료한 경우 |
| **BR-NOTI-002** | 모든 참여자 일정 제출 완료 | 방장 | 마지막 참여자가 일정 제출을 완료한 경우 |
| **BR-NOTI-003** | 여행 정보 변경 | 참여자 전원 | 방장이 여행방을 수정한 경우 (BR-TRIP-009·010) |
| **BR-NOTI-004** | 일정 확정 | 참여자 전원 | 방장이 후보 일정 중 하나를 최종 일정으로 확정한 경우 (BR-TRIP-007) |
| **BR-NOTI-009** | 일정 확정 취소 | 참여자 전원 | 방장이 확정 일정을 취소한 경우 (BR-TRIP-007) |
| **BR-NOTI-005** | 일정 업데이트 리마인드 (스케줄러) | 전체 사용자 | 매월 1일·15일 정기 발송 — **wave 4, MVP Out** |

## 알림 문구·랜딩 화면

**BR-NOTI-001 · 참여 완료**
> OO님이 여행방에 참여했어요! 참여 현황을 확인해보세요.

랜딩 화면: 여행방 상세

**BR-NOTI-002 · 모든 참여자 일정 제출 완료**
> 모든 참여자의 일정이 제출되었어요! 추천 일정을 받아보세요.

랜딩 화면: 여행방 상세

**BR-NOTI-003 · 여행 정보 변경**
> 여행 정보가 변경되었어요. 변경된 내용을 확인해보세요.

랜딩 화면: 여행방 상세 · `BR-USER-005` Off 시 미발송

**BR-NOTI-004 · 일정 확정**
> 여행 일정이 확정되었어요! 확정된 일정을 확인해보세요.

랜딩 화면: 여행방 상세 · `BR-USER-005` Off 시 미발송

**BR-NOTI-009 · 일정 확정 취소**
> 확정된 여행 일정이 취소되었어요. 다시 일정을 조율해보세요.

랜딩 화면: 여행방 상세 · `BR-USER-005` Off 시 미발송

**BR-NOTI-005 · 일정 업데이트 리마인드** (wave 4 — MVP Out)
> N월 일정을 업데이트해보세요. 더 정확한 여행 일정을 추천받을 수 있어요.

랜딩 화면: 일정 관리
