# Notification (알림) 비즈니스 규칙

> NotebookLM 기획 자료 정리본.

| 규칙 ID | 규칙명 | 조건 | 동작 | 위반 시 (에러/제약) |
| :--- | :--- | :--- | :--- | :--- |
| **BR-NOTI-001** | 리마인드 메시지 발송 | 방장이 미응답자를 확인할 때 | 방장은 시스템에서 생성된 리마인드 메시지를 카카오톡 등을 통해 공유할 수 있음 | `[기획 자료에 없음]` |
| **BR-NOTI-002** | 상태 변화 알림 | 여행 일정 조율 상태가 변경될 때 | 일정 확정, 새로운 참여자 응답 등의 이벤트를 알림으로 안내함 | 알림 설정 Off 시 수신 불가 |

| 규칙 ID | 이벤트 | 동작 |
| :--- | :--- | :--- |
| **BR-NOTI-003** | 참여 | `OO님이 ‘{여행방명}’에 참여했습니다` |
| **BR-NOTI-004** | 추천 갱신 | `‘{여행방명}’의 추천 일정을 확인해보세요` |
| **BR-NOTI-005** | 일자 변경 | 방장이 여행 일자 변경 시 멤버 알림 |
