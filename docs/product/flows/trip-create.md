# 여행방 생성 플로우

> NotebookLM · `trip-create`

- **목적**: 여행 조율을 위한 새로운 여행방 생성
- **액터**: 방장 (총대)
- **사전 조건**: 사용자가 로그인된 상태 (BR-USER-001)
- **단계**:
  1. 홈 화면에서 새 여행방 생성 버튼 클릭
  2. 여행방 이름 입력
  3. 여행 희망 기간(전체 날짜 범위) 선택
  4. 희망 여행 일수(n박 m일) 선택 (BR-TRIP-008)
  5. 예상 참여 인원 설정
  6. 만들기 버튼 클릭
- **성공 종료 조건**: 여행방이 생성되고 초대 링크 공유 화면으로 이동
- **예외 / 분기**: 필수 항목 누락 시 버튼 비활성화 (BR-TRIP-001)
- **MVP 포함 여부**: In
