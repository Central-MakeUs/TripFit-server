# 일정 확정 플로우

> NotebookLM · `trip-confirm`

- **목적**: 취합된 일정을 바탕으로 최적의 여행 날짜 확정
- **액터**: 방장 (총대)
- **사전 조건**: 모든(또는 주요) 참여자가 일정 입력을 완료함
- **단계**:
  1. 여행방 상세의 추천 결과 섹션 확인
  2. 시스템이 추천한 TOP 3 후보 일정과 추천 근거 검토 (BR-TRIP-005)
  3. 특정 후보의 일정 확정하기 버튼 클릭 또는 직접 날짜 선택
  4. 확정 확인 모달에서 최종 승인 (BR-TRIP-007)
- **성공 종료 조건**: 여행방 상태가 확정으로 변경되고 참여자들에게 알림 발송 (BR-NOTI-002)
- **예외 / 분기**: 권한 없는 사용자 접근 시 버튼 미노출 (BR-TRIP-007)
- **MVP 포함 여부**: In
