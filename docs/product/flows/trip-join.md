# 여행방 참여 플로우

> NotebookLM · `trip-join`

- **목적**: 공유받은 링크를 통해 여행방에 참여하고 일정 응답 시작
- **액터**: 참여자
- **사전 조건**: 방장으로부터 초대 링크 또는 코드를 수신함
- **단계**:
  1. 초대 링크 클릭 또는 참여 코드 입력
  2. (앱 미설치 시) 웹 뷰 또는 딥링크를 통해 여행방 상세 진입 (BR-USER-002)
  3. 여행방의 기본 정보(이름, 기간 등) 확인
  4. 내 일정 입력하기 버튼 클릭
- **성공 종료 조건**: 참여자 목록에 추가되고 일정 입력 화면으로 전환
- **예외 / 분기**: 유효하지 않은 코드/링크인 경우 에러 메시지 노출
- **MVP 포함 여부**: In
