# 일정·조건 입력 플로우

> NotebookLM · `schedule-edit`

- **목적**: 개인의 여행 가능 시간 및 근무/연차 조건 입력
- **액터**: 참여자 (방장 포함)
- **사전 조건**: 여행방에 참여 완료된 상태
- **단계**:
  1. 근무 형태(규칙/불규칙) 및 기본 근무 요일/시간 입력
  2. 여행 당 최대 연차 사용 가능 일수 등 연차 조건 설정 (BR-TRIP-006)
  3. 캘린더 화면에서 날짜별 오전/오후/저녁 단위로 가능/불가/미정 상태 선택 (BR-TRIP-002, BR-TRIP-003)
  4. (선택) 외부 캘린더 연동을 통해 기존 일정 불러오기 — MVP `[제안]`, `mvp.md` 참고
  5. 저장 또는 일정 제출하기 클릭
- **성공 종료 조건**: 입력된 데이터가 그룹 달력에 반영됨
- **예외 / 분기**: 일정 저장 전 이탈 시 변경사항 폐기 안내 팝업 노출
- **MVP 포함 여부**: In (캘린더 연동 단계는 `[제안]`)
